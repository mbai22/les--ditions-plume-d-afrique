# les--ditions-plume-d-afrique
Les Éditions Plume d'Afrique sont une maison d'édition indépendante qui place l'excellence littéraire et l'humain au cœur de son action.  Nous croyons que chaque histoire mérite d'être racontée, que chaque voix mérite d'être entendue. C'est pourquoi nous accompagnons nos auteurs avec passion, , de l'écriture à la diffusion.
